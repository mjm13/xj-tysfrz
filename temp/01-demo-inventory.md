# 01 — 原始 Demo 梳理

参照目录：`docs/原始demo/`  
主入口：`页面主页.html`（与 `platform-v2-home.html` 结构基本一致，可视为同版首页）

## 1. 设计规范（全站共用）

来源：各 HTML 内联 `:root` CSS 变量（注释称遵循 `design-spec-light`）

| 类别 | 要点 |
| --- | --- |
| 主色 | `#2563EB` / `#478EF7` |
| 中性色 | 文本 `#1D2129` / `#4E5969` / `#86909C`，背景 `#FAFAFB` / `#FFFFFF` / `#F2F3F5` |
| 布局 | 顶栏 64px sticky；内容区 max-width 1440px、padding 24px |
| 组件 | 数据卡片 8px 圆角 + shadow-1/2；段落标题左侧 4px 主色条；面包屑 + page-head |
| 字体 | PingFang SC / Microsoft YaHei，Body 14px |

## 3. 页面清单

### 3.1 平台总览

| 文件 | 标题 | 核心内容 |
| --- | --- | --- |
| `页面主页.html` | 高校综合身份数据平台 | 面包屑、统计头、7 功能导航卡、综合数据视图（人员总览 + 权限饼图 + 4 权限卡） |
| `platform-v2-home.html` | 同上 | 与主页同构（备用/迭代版） |

**首页 Mock 数据**：4 类人员（教职工/学生/校内服务/校外临时）、4 类权限（进校/校园网/校园卡/图书馆），前端 JS 联动饼图与权限卡。

### 3.2 身份管理模块（顶栏下拉）

| 文件 | 模块 | 模块内左侧栏 | 主要视图/能力 |
| --- | --- | --- | --- |
| `m1-basic-identity.html` | 人员基础身份 | 人员管理 + 本模块子菜单 | **基础数据**（列表/筛选/详情抽屉）、**变更记录**；侧栏链到 ETL |
| `m2-classification.html` | 人员分类身份 | 5 模块互链 + 管理入口 | **分类体系树**、**已映射人员**、**待映射**、**变更记录**；导入导出 Word/Excel/PDF |
| `m2-classification-admin.html` | 分类管理（子页） | 分类身份上下文 | **树维护**、**映射规则**、**花名册**、**变更**（hash: `#tree` `#mapping` `#roster` `#changes`） |
| `m3-position.html` | 人员岗位身份 | 5 模块互链 | **岗位数据**（只读）、**来源映射**；标注只读 |
| `m7-custom-tags.html` | 自定义标签身份 | 标签 + 跨模块链 | **标签组管理**、**组内人员** |
| `m4-org.html` | 组织机构体系 | 5 模块互链 | **组织树**（可编辑）、**来源映射**、**变更记录**；含来源映射 badge |

## 4. Demo 布局模式（实现时要复用）

```
┌─ Topbar（64px，全站） ─────────────────────────────┐
│ Logo | 主页 | 身份管理▾ | 权限 | 数据查询 | 用户 │
└──────────────────────────────────────────────────┘
┌─ Module Layout（业务页） ──────────────────────────┐
│ ┌ Sidebar ─┐ ┌ Main ────────────────────────────┐ │
│ │ 模块导航  │ │ 面包屑 + page-head + view 切换   │ │
│ │ 子菜单    │ │ 筛选条 / 树+表 / 图表 / 抽屉     │ │
│ └──────────┘ └──────────────────────────────────┘ │
└──────────────────────────────────────────────────┘
┌─ Footer ─────────────────────────────────────────┐
└──────────────────────────────────────────────────┘
```

首页（`/`）**无**模块内左侧栏，仅 Topbar + Main + Footer。

## 5. Demo 未覆盖

- **登录页**：Demo 目录无独立登录 HTML（产品假定已登录直接进入业务页）
- **真实鉴权 / RBAC**：无
- **后端 API**：纯静态 + 内联 Mock
